Manual QA Checklist

  • Test all pages & navigation flows
  • Click every button, link, and menu item
  • Validate forms (required fields, errors, success)
  • Responsive testing (desktop/tablet/mobile)
  • Cross-browser (Chrome/Firefox/Edge if possible)
  • Report issues with: page URL, steps, expected vs actual, screenshot

Tip: I can deliver findings in a Google Sheet or a simple PDF report.
